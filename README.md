 📌Project Overview

The API Integration and Data Visualization Project is a Python-based application that fetches real-time data from a public API and visualizes it using data visualization libraries. The project demonstrates how external data sources can be integrated into Python applications and transformed into meaningful insights through graphical representation.

In this project, weather data is retrieved from the OpenWeatherMap API for multiple cities. The data is processed and analyzed using Pandas, and visualizations are created using Matplotlib and Seaborn.

This project was developed as part of an internship task to demonstrate API integration, data processing, and visualization techniques in Python.

🚀 Features

✔ Fetch real-time data from a public API

✔ Process and organize data using Python

✔ Store retrieved data in a structured format

✔ Generate graphical visualizations of the data

✔ Display multiple charts as a simple visualization dashboard

✔ Compare weather parameters such as temperature and humidity across cities

🛠 Technologies Used

* Python 3

* API Integration using Requests

* Data Processing using Pandas

* Data Visualization using Matplotlib

* Statistical Visualization using Seaborn
